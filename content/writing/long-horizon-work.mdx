Long-horizon work is where agents become interesting and difficult.

It is easy to automate a short task when all the context is present. It is harder to keep a goal alive across days, shifting inputs, partial approvals, and new information that should change the plan.

The useful pattern is a sequence of commitments. The agent proposes the next step, gathers evidence, records why it changed direction, and pauses when the cost of being wrong becomes meaningful.

This makes the agent less theatrical and more operational. It becomes a system for carrying context forward, not a system for pretending uncertainty disappeared.
