## Context

Useful agents need more than a vector search box. They need a memory system that understands freshness, authority, contradictions, and the difference between a source and an interpretation.

This experiment treats company knowledge as a living substrate. Documents are broken into claims, linked back to sources, and exposed to agents with explicit provenance.

## Shape

The interface is intentionally quiet: a small review queue, a source graph, and a way to promote useful summaries into durable memory.

The interesting work is in the maintenance loop. The system asks for human judgment where ambiguity matters, then keeps those judgments available for future workflows.
