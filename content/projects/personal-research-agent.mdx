## Context

Research is rarely a single answer. It is a loop of questions, source quality, conflicting claims, and decisions about what is worth remembering.

This agent helps run that loop without hiding the messy parts. It collects sources, extracts candidate claims, asks for missing context, and produces a brief with unresolved questions left visible.

## Shape

The system is small by design. It keeps a local library of sources, notes, and recurring topics, then uses that memory to make future research faster.

The most useful feature is restraint: the agent is allowed to say that the available evidence is not enough.
